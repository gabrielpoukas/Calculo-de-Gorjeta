Descrição do Projeto

Este é um aplicativo de utilidade prática que calcula a gorjeta por pessoa e o valor total a ser pago, com base em três inputs diferentes: valor da conta, porcentagem da gorjeta e número de pessoas.

O projeto foi desenvolvido para consolidar o domínio de validação de formulário e lógica matemática no JavaScript.

✨ Tecnologias Utilizadas

Tecnologia	Função Principal
HTML5	Estrutura semântica e campos de entrada (<input type="number">).
CSS3	Layout Mobile-First, Flexbox para centralização e design dos inputs.
JavaScript (ES6+)	Lógica de cálculo, validação e manipulação do DOM (querySelector).

Funcionalidades e Habilidades Demonstradas

Este projeto solidificou habilidades cruciais de desenvolvimento Front-end:

    Leitura de Múltiplos Inputs: O JavaScript lê e converte corretamente valores de três campos diferentes (.value e parseFloat).

    Validação Essencial: Implementação de uma estrutura if/else para garantir que os cálculos só prossigam com números válidos e positivos (evitando erros de divisão por zero ou NaN).

    Lógica Financeira: Aplicação da matemática correta para isolar o valor da gorjeta e do total por pessoa.

    Formatação: Uso do método .toFixed(2) para exibir todos os resultados com duas casas decimais, garantindo o formato monetário (R$ 0.00).
